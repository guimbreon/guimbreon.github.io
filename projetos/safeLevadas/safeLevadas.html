safeLevadas

O projeto safeLevadas visa criar uma aplicação em Python para encontrar os percursos mais rápidos entre dois pontos na rede de levadas da Madeira, ajudando nas operações de resgate de caminhantes.

Tecnologias Usadas

  1. Python

Quando foi feito:

Início: 2023

Fim: 2024

Onde posso

encontrar o projeto?

Descrição

O safeLevadas é um software projetado para apoiar as equipas de resgate na determinação do percurso mais rápido entre dois pontos geolocalizados na rede de levadas da Madeira. Este sistema é essencial para auxiliar na rápida resposta a chamadas de socorro de caminhantes perdidos ou em perigo.

Funcionalidades

O programa recebe dois ficheiros de entrada:

  1. Rede de levadas: inputFile1.txt
  2. Pares de estações para calcular os percursos: inputFile2.txt

Gera um ficheiro de saída:

  1. Resultados dos percursos calculados: outputFile.txt

Como se executa o programa?

O programa deve ser executado no terminal da seguinte forma:

[guimbreon@ThreatForge]:/home$ python safeLevadas.py inputFile1.txt inputFile2.txt outputfile.txt

O que são os ficheiros de entrada?

  • inputFile1.txt: Ficheiro com a rede de levadas.
  • inputFile2.txt: Ficheiro com pares de estações para os quais se deseja calcular os percursos mais rápidos.

O software é composto pelos seguintes módulos:

  • constants.py: Define as constantes necessárias.
  • dateTime.py: Contém funções para lidar com formatos e operações com datas e tempos.
  • infoFromFiles.py: Fornece funções para ler informações de ficheiros.
  • planning.py: Contém funções para realizar a determinação dos percursos mais rápidos.
  • infoToFiles.py: Oferece funções para escrever informações em ficheiros.
  • safeLevadas.py: Programa principal que utiliza os módulos anteriores para realizar a aplicação safeLevadas.
