birthPlan

Descrição

birthPlan é um software projetado para apoiar a equipa de enfermagem no encaminhamento de grávidas para os serviços de maternidade. Ele será utilizado pela coligação de hospitais smartH para gerir a atribuição de assistência no parto a médicos especialistas e respetivas equipas.

Como funciona?

Recebe 3 ficheiros com informações de Entrada:

  1. Médicos;
  2. Calendário de assistências;
  3. Pedidos de assistências.

Gera 3 ficheiros Saida:

  1. Lista atualizada dos médicos;
  2. Calendário de assistências atualizado.

Como se executa o programa?

O programa deve ser executado no terminal da seguinte forma:

[guimbreon@ThreatForge]:/home$ python3 inputFile1.txt inputFile2.txt inputFile3.txt

Oque é cada inputeFile.py?

  • inputFile1.py: Ficheiro com a listagem dos médicos;
  • inputFile2.py: Ficheiro com a calendarização das assistências;
  • inputFile3.py: Ficheiro com a listagem dos pedidos de assistência.

O software é composto pelos seguintes módulos:

  • constants.py: Define as constantes necessárias
  • dateTime.py: Contém funções para lidar com formatos e operações com datas e tempos.
  • infoFromFiles.py: Fornece funções para ler informações de ficheiros.
  • planning.py: Contém funções para realizar a calendarização e atualização das estruturas de dados.
  • infoToFiles.py: Oferece funções para escrever informações em ficheiros.
  • refresh.py: Programa principal que utiliza os módulos anteriores para realizar a aplicação birthPlan.
