Description

birthPlan is software designed to support the nursing team in referring pregnant women to maternity services. It will be used by the smartH hospital coalition to manage the assignment of delivery assistance to specialist doctors and their respective teams.

How does it work?

It receives 3 input files with information:

  1. Doctors;
  2. Assistance schedule;
  3. Assistance requests.

Generates 3 output files:

  1. Updated list of doctors;
  2. Updated assistance schedule.

How to run the program?

The program should be run in the terminal as follows:

[guimbreon@ThreatForge]:/home$ python3 inputFile1.txt inputFile2.txt inputFile3.txt

What is each inputFile.py?

  • inputFile1.py: File with the list of doctors;
  • inputFile2.py: File with the assistance schedule;
  • inputFile3.py: File with the list of assistance requests.

The software consists of the following files:

Objects:

  • Schedule: objects representing schedules;
  • DateTime: objects handling dates and times;
  • Mother: objects representing mothers;
  • Doctors: objects representing doctors;
  • Lists: objects for list manipulation;
  • FileHandling: objects handling file operations;
  • infoToFiles: writes information to files.

Additional Files:

  • constants: the constants necessary for the project;
  • testing: the project's test file;
  • refresh: the file used to run the project!
