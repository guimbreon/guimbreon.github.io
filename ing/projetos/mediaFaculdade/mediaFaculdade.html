Description

Allows calculating the college average in a faster and easier way.

Website Features

Version symbols:

: Version 1

: Version 2

Registration and Login

  • Registration and Login Page: Allows users to create an account and log in.
    • Registration: Users can create an account by providing a username and password.
    • Login: Users can log in using their username and password.

Individual Grade Calculator

  • Assessment Input: Users can enter the number of assessment parts.
  • Dynamic Fields: Dynamic generation of fields for entering assessment values and percentages.
  • Final Grade Calculation: Calculation of the final grade based on the entered values and percentages.

Overall Average Calculator

  • File Import: Allows importing Excel files containing course data and respective grades.
  • Manual Input: Allows manual entry of courses, year, semester, ECTS, and grade.
  • Average Calculation: Calculation of the weighted average of the courses based on the ECTS.
  • Export Results: Ability to export results to an Excel file.

How to calculate the average?

Just open the index.html file (OR at this link) and access "Calculate total grade?".
