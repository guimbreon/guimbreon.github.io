Description

The Memory Game is an interactive game where players need to find pairs of matching cards. It is designed to be educational and fun, helping to improve players' memory.

How does it work?

To start the game, all cards are placed face down so they cannot be seen and are arranged randomly. Each player, on their turn, must flip over two cards and let everyone see them.

Then two things can happen:

  1. If the pair matches, the player should collect that pair, earning a point, and play again.
  2. If the pair does not match, the cards should be turned face down again, and it becomes the next player's turn.

The game ends when all pairs have been found or by clicking the end game button.

How to play the game?

The game runs directly in the browser. Just open the HTML file in any compatible browser (OR at this link).

Information you should know:

On the personal page, there is a table with your best times and a shop where you can buy different decks to play with based on the points awarded to each player.

Points are awarded as the player wins games and depending on the difficulty of each match (game time limit), which is chosen at the start of each game:

  • No limit: 1 Point
  • 10-minute limit: 2 Points
  • 7-minute limit: 4 Points
  • 3-minute limit: 7 Points
  • 1-minute limit: 10 Points

At the beginning of each match, you can also choose the game mode by clicking on its option:

  • Click: Play by clicking on the cards
  • Type: Play by typing the IDs of the cards in the text boxes

There is also a statistics page where you can see some information about played matches, such as the best global ranking with time and player name, and the best personal ranking with the top 10 times of each player.

The software is organized into the following pages:

  • extra: Where the extra files are (like the extra mini-game);
  • header: Where the header pages are;
  • images: Where the site images are stored;
  • index: Where the initial index page is;
  • game: game files, such as the board and menu;
  • script: JavaScript files;
  • sounds: sounds used in the game;
  • styles: CSS files;
  • videos: videos used in the game.
