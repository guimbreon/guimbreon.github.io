safeLevadas

The safeLevadas project aims to create a Python application to find the fastest routes between two points in Madeira's levadas network, aiding in hiker rescue operations.

Technologies Used

  1. Python

Project Timeline:

Start: 2023

End: 2024

Description

safeLevadas is software designed to support rescue teams in determining the fastest route between two geolocated points in Madeira's levadas network. This system is essential for assisting in the quick response to distress calls from lost or endangered hikers.

Features

The program accepts two input files:

  1. Levada network: inputFile1.txt
  2. Station pairs to calculate routes: inputFile2.txt

It generates one output file:

  1. Calculated route results: outputFile.txt

How to run the program?

The program should be executed in the terminal as follows:

[guimbreon@ThreatForge]:/home$ python safeLevadas.py inputFile1.txt inputFile2.txt outputfile.txt

What are the input files?

  • inputFile1.txt: File with the levada network.
  • inputFile2.txt: File with station pairs for which the fastest routes are to be calculated.

The software consists of the following modules:

  • constants.py: Defines necessary constants.
  • dateTime.py: Contains functions for handling date and time formats and operations.
  • infoFromFiles.py: Provides functions for reading information from files.
  • planning.py: Contains functions for determining the fastest routes.
  • infoToFiles.py: Offers functions for writing information to files.
  • safeLevadas.py: Main program that uses the previous modules to execute the safeLevadas application.
