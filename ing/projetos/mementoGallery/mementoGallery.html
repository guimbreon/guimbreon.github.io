Description

MementoGallery allows users to manage large photo collections intuitively and efficiently.

Website Features

Version symbols:

: Version 1

: Version 2

Photo Collection Management

  • Photo Import: Import photos from various sources (e.g., Instagram, Facebook, Flickr, or local disk).
  • Creation of Representative Subset: Generate a subset of photos based on user-defined criteria, such as maximum number of photos, quality, type of photo, emotional context, among others.
  • Interactive Refinement: Allows users to adjust criteria interactively to refine the final photo selection.

Photo Removal

  • Manual Removal: Possibility to remove photos from the collection with a simple click (e.g., via a "+" button).

Customizable Functionality

  • Additional Functionality: Extra features based on user needs analysis.

How to use the application?

Simply open the index.html file (OR visit the link) and start managing your photo collection.
